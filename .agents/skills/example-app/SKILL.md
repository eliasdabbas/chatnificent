Create a new app always making sure to adhere to the following guidelines:

- each app should be runnable  as a script: `use uv add --script pkg1 pkg2 pkg3 etc`
- Structure:
  - script requirements
  - triple-quoted module docstring
  - app code
- Doc string: should be extensive explaining why we are doing these things what else to explore and so on. This should be a self-contained how-to guide on how to create a specific app with chatnificent
- Check previous examples and be specific and detailed about the example you are explaining
